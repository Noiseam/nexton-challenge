
## Description

Challenge for Nexton, its a calculator.
done in [Nest](https://github.com/nestjs/nest)
you need node 18.14 at least to run it (havent really tried with previous versions)
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testing the endpoints

```bash
# swagger
$ /api

# post endpoint
$ /calculator
```

## Test

```bash
# unit tests
$ npm run test

```
